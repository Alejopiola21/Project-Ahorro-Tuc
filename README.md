# 🛒 Ahorro Tuc

**Ahorro Tuc** es una plataforma web inteligente diseñada específicamente para los habitantes de San Miguel de Tucumán y alrededores. Nuestra misión es ayudar a las familias tucumanas a combatir la inflación y ahorrar dinero en sus compras cotidianas comparando precios en tiempo real.

## 🌟 Propósito del Proyecto
En un contexto de constante variación de precios, saber dónde comprar puede significar un ahorro de miles de pesos al mes. **Ahorro Tuc** centraliza la información de las principales cadenas de supermercados de la provincia para que no tengas que recorrerlos físicamente.

## 🚀 Funcionalidades Principales

- **🔍 Buscador en Tiempo Real**: Buscá productos como leche, yerba o aceite y compará al instante los precios entre Coto, Carrefour, Jumbo, Vea, Disco y Día.
- **🥇 Indicador de Mejor Precio**: El sistema resalta automáticamente el supermercado más barato para cada producto individual.
- **🛒 Carrito Inteligente**: Armá tu lista de compras completa. Nuestra tecnología simula el total del ticket en cada cadena y te dice exactamente en qué supermercado gastarás menos por el total de tu lista.
- **💰 Cálculo de Ahorro Máximo**: Visualizá cuánto dinero evitás gastar eligiendo la opción ganadora.

## 🛠️ Stack Tecnológico

- **Frontend**: React + TypeScript + Vite. Diseño moderno con CSS Vanilla (Glassmorphism).
- **Backend**: Node.js + Express + TypeScript.
- **Base de Datos**: SQLite (Better-SQLite3) para persistencia rápida y eficiente.
- **Iconografía**: Lucide React.

## 🏗️ Estado del Proyecto
Actualmente el proyecto se encuentra en su fase de **MVP (Producto Mínimo Viable)** con:
- Estructura Full-Stack conectada.
- Lógica de carrito y comparativa funcional.
- Base de datos inicializada con productos semilla.
